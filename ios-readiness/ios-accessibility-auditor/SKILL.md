# iOS Accessibility Auditor

Audit the provided SwiftUI or UIKit code for accessibility best practices:
- Meaningful accessibility labels and hints
- Support for Dynamic Type (scalable fonts)
- Sufficient color contrast for text and interactive elements
- Proper focus management for VoiceOver

List any missing accessibility modifiers and suggest how to add them.
