# iOS Privacy Auditor

Audit the provided code or project description for iOS privacy requirements:
- Proper data handling and tracking
- Necessary entries for Privacy Manifests (PrivacyInfo.xcprivacy)
- Required `Info.plist` usage descriptions (e.g., Camera, Location, Photo Library)

Flag any missing descriptions or improper data practices.
