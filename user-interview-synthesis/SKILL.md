# User Interview Synthesis

**Purpose:** Extract actionable insights from user interviews.

When given raw interview transcripts or notes:
- Extract top 3 pain points with quotes and frequency.
- Highlight one surprising insight.
- Group and list feature requests.
- Assign a confidence level (High/Medium/Low) to each finding.
