# QA Test Generator

**Purpose:** Automatically create manual test scripts for QA.

When provided with a feature spec, generate a step-by-step QA checklist in plain English.
Include:
- 1 Happy path
- 3 Unhappy paths (edge cases, errors, invalid inputs)
