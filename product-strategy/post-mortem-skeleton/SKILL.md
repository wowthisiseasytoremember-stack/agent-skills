# Post-Mortem Skeleton

**Purpose:** Create a mini post-mortem from raw feedback.

When provided with feedback or an incident description, create a post-mortem containing:
- Root cause (one sentence)
- Preventive action
- Blast radius (estimated customers/systems affected)
- Priority score
