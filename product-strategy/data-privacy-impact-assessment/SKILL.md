# Data & Privacy Impact Assessment

**Purpose:** Assess privacy risk before shipping.

When given a feature description:
- List personal data collected and storage duration.
- Identify GDPR/CCPA/HIPAA risks.
- Check if a user consent mechanism is required.
- Provide a one-line Go/No-Go recommendation.
