# Risk Register Builder

**Purpose:** Generate a risk register for a program.

When given a program or project description, generate a risk register as a Markdown table with the following columns:
- Risk description
- Likelihood (1-5)
- Impact (1-5)
- Owner
- Mitigation one-liner
