# Demo Script Generator

**Purpose:** Prepare for effective feature demos.

When given a feature and audience:
- Start with the problem solved (1 sentence).
- Script 3 key "moments" (avoiding wasted clicks).
- Conclude with the business impact (not technical details).
- Keep the script within the specified time limit.
