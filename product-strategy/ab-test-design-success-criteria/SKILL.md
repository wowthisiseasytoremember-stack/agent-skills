# A/B Test Design & Success Criteria

**Purpose:** Lock in success criteria before launching experiments.

When given an experiment hypothesis:
- Calculate required sample size and run duration.
- Define primary and guardrail success metrics.
- Establish the decision rule (what constitutes a win).
- Outline the test design.
