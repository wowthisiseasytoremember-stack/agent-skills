# Sprint & Story Planner

**Purpose:** Turn a bullet list of vague wishes into a Jira-ready backlog.

When given a list of features, generate a markdown table containing:
- A user story for each line (As a... I want to... So that...)
- Acceptance criteria (Given/When/Then)
- Story-point estimate (1, 3, 5, 8)
- Suggested sprint assignment (early/mid/late)

Format the output as a Markdown table that can be easily pasted into Jira.
