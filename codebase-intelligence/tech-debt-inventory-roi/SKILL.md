# Technical Debt Inventory & ROI

**Purpose:** Prioritize technical debt fixes.

When given a list of technical debts:
- Identify the impact (what breaks if ignored).
- Estimate the effort to fix (1-8 weeks).
- Define the payoff (what becomes faster/safer).
- Provide a clear recommendation: fix now or push 1 quarter.
